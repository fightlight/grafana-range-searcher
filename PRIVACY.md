Privacy Policy for Grafana Range Searcher

Last updated: February 2026

1. DATA COLLECTION
   Grafana Range Searcher does NOT collect, transmit, or share any personal data or browsing information.

2. LOCAL STORAGE
   The extension stores the following data locally on your device only:
- Start time preference
- Time interval setting
- Current range position

This data never leaves your browser and is used solely to preserve your settings between sessions.

3. PERMISSIONS USED
- "tabs": Used to read the current tab URL and update it with new time range parameters. The extension only modifies Grafana-specific URL
  parameters (from/to timestamps).
- "storage": Used to save your preferences locally in Chrome storage.

4. THIRD-PARTY SERVICES
   This extension does not use any third-party services, analytics, or tracking.

5. CONTACT
   For questions about this privacy policy, contact: fightlight.nightsight@gmail.com

6. CHANGES
   Any changes to this policy will be reflected in the "Last updated" date above.
